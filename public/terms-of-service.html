Condiciones del Servicio

Estas condiciones regulan el acceso y uso de la web de Ashford Barberia, asi como la solicitud y gestion de reservas, comunicaciones operativas y servicios vinculados a nuestra actividad.

Ultima actualizacion: 20 de abril de 2026.

1. Objeto del servicio

La web permite consultar informacion del negocio y solicitar citas para servicios de peluqueria y barberia. La disponibilidad, precios, duraciones y condiciones pueden actualizarse en cualquier momento para reflejar la operativa real del establecimiento.

2. Reservas

La reserva se considera solicitada cuando el usuario completa el formulario y se registra correctamente en el sistema.
La confirmacion definitiva puede depender de revision interna, disponibilidad real o contacto posterior con el cliente.
Ashford Barberia podra rechazar o reprogramar solicitudes en caso de error tecnico, duplicidad, falta de disponibilidad o causas operativas justificadas.

3. Datos facilitados por el usuario

El usuario se compromete a proporcionar informacion veraz, actual y suficiente para gestionar la reserva. Si los datos son incorrectos o incompletos, la prestacion del servicio o las comunicaciones asociadas podrian verse afectadas.

4. Cancelaciones, cambios y no asistencia

Los cambios y cancelaciones deben solicitarse con la mayor antelacion posible. La no asistencia o los cambios reiterados de ultima hora podran afectar a futuras reservas si alteran de forma relevante la organizacion del servicio.

5. Precios y contenido de la web

Intentamos mantener la informacion de la web actualizada, pero puede haber cambios en servicios, horarios, duraciones o precios. En caso de discrepancia evidente o error material, prevalecera la informacion confirmada directamente por el negocio.

6. Uso permitido

No esta permitido usar la web para enviar informacion falsa, suplantar a terceros o realizar reservas fraudulentas.
Tampoco se permite interferir con el funcionamiento tecnico del sitio o intentar acceder a zonas no autorizadas.

7. Responsabilidad

Ashford Barberia no sera responsable de interrupciones temporales, incidencias ajenas a su control razonable, errores derivados de servicios de terceros o problemas de conectividad que afecten a la disponibilidad puntual de la web.

En todo caso, se actuara con diligencia razonable para restaurar el servicio y atender incidencias cuando sea posible.

8. Privacidad y comunicaciones

El tratamiento de datos personales se regula por nuestra Politica de Privacidad. Algunas notificaciones operativas podrian gestionarse mediante proveedores externos, incluida mensajeria empresarial, cuando resulte necesario para el funcionamiento del servicio.
